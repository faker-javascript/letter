## Install

```
$ npm install --save @fakerjs/letter
```

## Usage

```js
import letter from '@fakerjs/letter';

letter();
//=> z

letter({'casing': 'upper'});
//=> Z

letter({'casing': 'upper', 'locale': 'ru_RU'});
//=> Ð¯
```

## Tests

Run tests

```
npm run test
```
